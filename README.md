# lap4-url-shortener

## Installation & usage

-   Clone or download code:
    -   cd into the folder
    -   in terminal run:
        1. pipenv shell
        2. pip install pipenv
        3. pipenv install
    -   To run development mode:
        -   pipenv run dev
        -   open http://localhost:5000 to view it in your browser
        -   remember to refresh the page upon code changes
